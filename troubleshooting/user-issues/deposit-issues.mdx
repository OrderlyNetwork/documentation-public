---
title: "Stuck/delayed deposits"
description: "How to diagnose and resolve delayed or stuck deposits using the Dune dashboard and support channels."
---

Deposits do not reflect immediately in user's account.

If users are having issues with deposits please do the following:

<Steps>
  <Step title="First Step">
    Check our [Dune dashboard](https://dune.com/orderly_network/orderly-support-dashboard) and check
    if deposit went through to any builder. Sometimes users deposit on one builder and expect the deposit to be reflected on another builder's site.

    If the deposit is seen on the Dune dashboard, the funds are in the user's account and most likely a front-end issue if the balance is not displaying.

    <Note>
    Make sure to refresh the dashboard before checking.
    </Note>

    <Note>
    Deposits times vary based on chain. Polygon deposit requires 500 block confirmations which takes about 20 minutes.
    </Note>

  </Step>

  <Step title="Second Step">
If that wasn't successful, please search the user's wallet address on [LayerZero explorer](https://layerzeroscan.com/).

A message appears with Orderly as the destination chain for any deposits we have received.

- If a message is in-flight, the deposit is processing.
- If a message is delivered, the deposit was processed.
- If no message appears, Orderly has not received or processed the deposit. Please proceed to next step.

 </Step>

  <Step title="Third Step">
  If that wasn't successful, please send a message to Orderly with

- User's address
- Deposit amount
- Approximate time of deposit.

 </Step>
</Steps>
