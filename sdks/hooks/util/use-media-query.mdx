---
title: "useMediaQuery"
description: "Hook to evaluate a CSS media query and return a boolean for responsive layouts."
---

Evaluate a CSS media query via JavaScript.

- [Tech docs](/sdks/tech-doc/modules/orderly_network_hooks#usemediaquery)

### Example

```tsx
const isTablet = useMediaQuery("(max-width: 768px)");

return <div>{isTablet ? "is phone or tablet" : "is desktop"}</div>;
```
