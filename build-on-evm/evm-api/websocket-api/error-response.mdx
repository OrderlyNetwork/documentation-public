title: "Error Response"
---

Example response if there is an error.

You can use the `id` field in response to map the error response with the request. 

```json
{
    "id": "clientID7",
    "event": "subscribe",
    "success": false,
    "ts": 1614141150601,
    "errorMsg": "invalid symbol SPOT_WOO_USDC.e"
}
```
