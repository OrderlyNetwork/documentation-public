---
title: "Traders Open Interest"
---

Push interval: every 1m

```json
{
    "id": "clientID11",
    "topic": "traders_open_interest",
    "event": "subscribe"
}
```

**Parameters**

| Name   | Type   | Required | Description               |
| ------ | ------ | -------- |---------------------------|
| id     | string | Y        | id generate by client     |
| event  | string | Y        | `subscribe`/`unsubscribe` |
| topic  | string | N        | `traders_open_interest`   |

<RequestExample>
```json Response
{
   "id":"clientID11",
   "event":"subscribe",
   "success":true,
   "ts":1682242408023
}
```
</RequestExample>
