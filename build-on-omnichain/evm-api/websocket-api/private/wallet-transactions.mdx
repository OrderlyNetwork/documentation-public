---
title: "Wallet Transactions"
---

Push interval: real-time push

```json
{
    "id": "clientID3",
    "topic": "wallet",
    "event": "subscribe"
}
```

**Parameters**

| Name   | Type   | Required | Description                              |
| ------ | ------ | -------- | ---------------------------------------- |
| id     | string | Y        | id generate by client                    |
| event  | string | Y        | `subscribe`/`unsubscribe`                |
| topic  | string | Y        | `wallet`                                |

```json Response

    "success": true,
    "ts": 1609924478533
}
```
